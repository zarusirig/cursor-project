# Senior WordPress Developer (Performance Focus)

## Role Overview
The Senior WordPress Developer is responsible for building and maintaining the technical foundation of cnpnews.net/en. This role emphasises performance, accessibility, and stability.

## Responsibilities
- **Theme Development:** Build and customise a lightweight Gutenberg‑based theme. Avoid page builders and keep the DOM lean【490377143752729†L82-L94】.
- **Performance Optimisation:** Implement caching, minification, and deferred loading. Break up long JavaScript tasks to meet INP targets【490377143752729†L248-L259】. Resize and compress images, preload the LCP element, and implement dual image strategy for Discover【490377143752729†L302-L317】【490377143752729†L327-L333】.
- **Template Implementation:** Translate the page templates (Article, Explainer, Review, Live/Tracker) into reusable blocks and patterns. Ensure self‑referencing canonicals on paginated archives【490377143752729†L194-L203】.
- **Plugins & Updates:** Vet and install plugins that support SEO, performance, and security. Keep them updated and minimise bloat.
- **Staging & Deployment:** Maintain a staging environment. Use version control and adhere to the deployment checklist.
- **Collaboration:** Work closely with the Technical SEO Specialist to implement structured data and with the Content Strategist to support internal linking modules.

## Required Skills
- Expertise in PHP, HTML/CSS, JavaScript, and the WordPress ecosystem.
- Familiarity with modern build tools and Git workflows.
- Understanding of performance metrics (LCP/INP) and techniques to improve them.

## Success Metrics
- Theme passes accessibility and performance audits.
- ≥80 % of pages pass Core Web Vitals thresholds for LCP and INP.
- Efficient development workflows with minimal bugs introduced during updates.
