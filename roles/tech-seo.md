# Technical SEO Specialist (The Architect)

## Role Overview
The Technical SEO Specialist is responsible for the discoverability, indexability, and technical health of cnpnews.net/en. This person ensures that our WordPress site adheres to best practices for sitemaps, canonicals, structured data, and performance.

## Required Skills
- Strong understanding of technical SEO principles and WordPress architecture.
- Familiarity with schema.org standards, XML sitemaps, and robots directives.
- Ability to analyse web performance metrics and diagnose issues.

## Success Metrics
- Sitemaps remain clean with no indexation errors.
- Canonical tags correctly implemented across all archives.
- Structured data free of errors; GSC Enhancement reports clear.
- Page performance metrics (LCP/INP) meet targets.
